## 🎯 QISQACHA XULOSA

**Avtojon** - O'zbekistondagi yuk tashish kompaniyalari uchun mo'ljallangan SaaS platforma.

### Asosiy modullar:
| Modul | Maqsad |
|-------|--------|
| 📦 Dashboard | Reyslar, shofyorlar, xarajatlar, foyda |
| 🚛 Fleet | Avtopark tracking, yoqilg'i, moy, shina |

### Foydalanuvchilar:
- 👔 **Biznesmen** - Kompaniya egasi
- 🚛 **Shofyor** - Haydovchi (mobil ilova)
- 🔧 **Super Admin** - Tizim administratori

### Texnologiyalar:
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** MongoDB + Redis
- **Mobile:** Capacitor (Android/iOS)
- **AI:** Groq Whisper + LLM (ovozli kiritish)
