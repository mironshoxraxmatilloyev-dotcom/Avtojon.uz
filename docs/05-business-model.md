## 📈 UNIT ECONOMICS

### Asosiy ko'rsatkichlar

| Metrika | Qiymat | Maqsad |
|---------|--------|--------|
| **CAC** (Customer Acquisition Cost) | $5 | <$10 |
| **ARPU** (Average Revenue Per User) | $10/oy | $15/oy |
| **LTV** (Lifetime Value) | $200 | $300 |
| **LTV/CAC** | 40x | >20x |
| **Payback Period** | 0.5 oy | <3 oy |
| **Gross Margin** | 85% | >80% |
| **Churn Rate** | 3%/oy | <5%/oy |

### LTV Hisoblash:
```
LTV = ARPU × Gross Margin × (1 / Churn Rate)
LTV = $10 × 0.85 × (1 / 0.03)
LTV = $283
```
